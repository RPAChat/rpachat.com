Get Started with Only a Few Lines of Code

Read Documentation →

It only takes 4 steps to start developing with RPAChat using our open source partner, Wechaty.

  • Obtain an RPAChat token.
  • Set the environment to your desired platform, like WhatsApp, WeChat, or WeCom, with the puppet variable.
  • Scan the QR to connect your device to RPAChat.
  • After login, use the event driven SDK to send and receive messages easily.
import { WechatyBuilder } from 'wechaty'

const wechaty = WechatyBuilder.build( { // get a Wechaty instance
  puppet: 'wechaty-puppet-service',
  puppetOptions: { token } // insert your RPAChat token
  } )
wechaty
  .on('scan', (qrcode, status) => console.log(`Scan QR Code to login: ${status}\nhttps://wechaty.js.org/qrcode/${encodeURIComponent(qrcode)}`))
  .on('login',            user => console.log(`User ${user} logged in`))
  .on('message',       message => console.log(`Message: ${message}`))
wechaty.start()
